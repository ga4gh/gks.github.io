# gks.github.io

The landing page for getting started with GA4GH Genomic Knowledge Standards

## Development

```shell
git clone https://github.com/ga4gh/gks.github.io
cd gks.github.io
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Docs

To run the docs locally:

```shell
mkdocs serve
```

Then go to <http://127.0.0.1:8000/>
